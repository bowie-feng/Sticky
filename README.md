# 便条贴 (Sticky)

一款 macOS 桌面便条贴应用，让你在桌面上随手记录待办事项。

## 功能

- 📝 **桌面便条** — 在桌面创建彩色便条，记录待办事项和备忘
- 🎨 **多彩主题** — 8 种预设颜色（黄、绿、蓝、粉、紫、橙、白、青）
- 📌 **固定到桌面** — 将便条固定在指定位置，防止误拖
- 🔍 **字体调节** — 支持 10pt ~ 36pt 多种字体大小
- 💧 **透明度调节** — 支持 100% ~ 35% 多档透明度
- 💾 **自动保存** — 所有内容自动持久化，重启不丢失
- 🌓 **深色模式** — 适配 macOS 浅色 / 深色模式

## 系统要求

- macOS 14.0+

## 安装

### 从源码编译

```bash
# 编译
./build.sh

# 编译并安装到 /Applications
./build.sh --install
```

### 手动安装

编译后将 `.build/Sticky.app` 拖入 `/Applications` 文件夹即可。

## 使用

- **新建便条**：菜单栏 `文件 → 新建便条` 或快捷键 `⌘N`
- **编辑内容**：直接点击便条即可输入文字
- **移动便条**：拖拽便条背景任意位置
- **调整大小**：拖拽右下角
- **右键菜单**：更改颜色、透明度、字体大小、固定/取消固定
- **删除便条**：右键 `删除便条` 或按 `Esc` 键
- **关闭所有**：菜单栏 `文件 → 关闭所有便条` 或快捷键 `⌘K`

## 技术栈

- SwiftUI + AppKit
- Swift Package Manager
- 纯原生 macOS 实现，无第三方依赖
